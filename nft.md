# ■ 휴대용 채굴기(NFT, 아이템, 코인 등)

### 1. 개요

* 특허 출원 신청 : 휴대용 NFT 하드월렛 플레이어 뷰어(2021년 11월)
* 명칭 : 게임 아이템채굴기기(하드웨어 및 소프트웨어로 구성)   &#x20;
* 하드웨어 개발 :  디스플레이 개발(5.4인치 OLED 디스플레이 +도킹바 )&#x20;
* 안드로이드 OS 개발
* 펌웨어개발&#x20;
* 핵심키워드 THIN (얇은 디스플레이 뷰어)&#x20;
* 휴대용 ITEM 마이닝 시스템 탑재
